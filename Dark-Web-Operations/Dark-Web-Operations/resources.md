## 🛠️ Tools Used

### TOR Browser
- **Purpose:** Secure, anonymous access to the dark web
- **Source:** [TOR Project Official Site](https://www.torproject.org)
- **Usage:** Used to navigate .onion sites safely during course activities

### Steganography
- **Purpose:** Extracting hidden information concealed within image files
- **Usage:** Used during the Dark Web Operations Challenge to extract 
  intelligence hidden inside an image file
- **Key Skill:** Identifying and retrieving covert data embedded in 
  non-obvious locations — a critical threat intelligence skill

### CyberChef
- **Purpose:** Decoding, decrypting, and analyzing data
- **Source:** [CyberChef - GCHQ](https://gchq.github.io/CyberChef/)
- **Usage:** Used to decode numerical data extracted from the 
  steganography challenge
- **Key Skill:** Data analysis and decoding — essential for SOC analysts 
  and threat intelligence roles

## 💡 Personal Notes on Tools

### Steganography + CyberChef Workflow
During the course capstone challenge, I used the following workflow:
1. Received an image file containing hidden intelligence
2. Used steganography techniques to extract concealed data from the image
3. Retrieved encoded numerical data from the image
4. Used CyberChef to decode the numerical data and reveal the intelligence
5. Documented findings as part of the Dark Web Intelligence Analyst (DWIA) scenario

> 💡 This workflow is directly applicable to real-world threat intelligence 
> and digital forensics investigations — a valuable skill to highlight in 
> job applications and interviews.
